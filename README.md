# ControlBiblioteca
Proyecto de práctica para aprender el uso básico de Git y control de versiones.

## Archivos del proyecto

- Libro.txt
- Autor.txt
- Usuario.txt
- Prestamos.txt

## Temas practicados

- Inicialización de repositorio
- git init
- git status
- git add
- git commit
- git log
- git log --oneline
- git diff
- git restore

## Objetivo

Simular el control de versiones de un sistema básico de biblioteca utilizando Git.
